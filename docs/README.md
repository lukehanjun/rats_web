# RATs Project Website

This directory is a dependency-free GitHub Pages site. Configure GitHub Pages to
serve from the repository's `rats_web/docs/` directory.

## Video Assets

The result gallery uses these files from `assets/videos/`:

- Real world: `close_drawer.mp4`, `cube_transfer.mp4`, `open_drawer.mp4`
- Simulation: `libero.mp4`, `molmospaces.mp4`, `robosuite.mp4`

The pipeline overview remains a designed placeholder until
`assets/videos/pipeline-overview.mp4` is ready.
