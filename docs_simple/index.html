Abstract

Current agentic robot systems can write executable Code-as-Policy programs, observe feedback, and revise behavior across multiple attempts, but they remain largely task-driven: reusable skills are acquired only after explicit instructions. We study Playful Agentic Robot Learning, where an embodied coding agent uses self-directed play as a continual skill-learning stage before downstream tasks arrive.

We introduce RATs, a Robotic Agent Team designed for play-time skill acquisition. During play, RATs proposes novel yet learnable exploratory tasks, plans and executes robot-code policies, verifies intermediate progress, diagnoses failures, retries with feedback, and distills successful executions into a persistent code skill library.

Method

RATs organizes play as a closed loop. A Task Proposer chooses an objective that is new enough to be informative but close enough to the current skill library to remain learnable. The Robotic Agent Team then plans, writes, executes, verifies, and revises a policy. Successful attempts update the skill library; unsuccessful attempts update failure memory.

1. Propose

Generate scene-grounded objectives and select one near the competence frontier.

2. Execute

Plan, write, execute, verify, diagnose, and retry robot-code policies.

3. Accumulate

Distill successful behavior into learned skills and failures into reusable lessons.

Evaluation

The final paper reports that play-learned skills improve held-out downstream tasks without finetuning the underlying model. The evaluation artifacts for these comparisons are not included in the local repository snapshot; the values below are paper-reported.

Inside Play

A locally available 50-iteration MolmoSpaces play run shows how RATs diversifies practice and accumulates knowledge. The proposer covers seven interaction families and more than 30 object categories across 49 saved proposal records. Over the same run, the learned library grows from 6 helpers at iteration 10 to 27 helpers at iteration 50.

Choosing a learnable objective

At iteration 15, the proposer selects a tissue-box lift with a Goldilocks score of 0.6236. The task is meaningfully different from recent attempts, but remains grounded in emerging grasp and motion capabilities. This is the intended role of play: practice objectives that are challenging enough to teach and grounded enough to solve.
